Multi-predictor:A Web Application for Machine Learning Predictions


Our project aims to deploy machine learning models through a web application built using the Flask web framework. The website provides users with the ability to interact with trained models for two different datasets: the Ames Housing dataset and the Titanic dataset.


Purpose


The project aims to demonstrate the end-to-end process of deploying machine learning models through a web application, providing users with access to predictive analytics in an intuitive and accessible manner.
